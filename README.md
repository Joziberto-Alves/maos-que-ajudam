# 🌱 Mãos que Ajudam

Projeto desenvolvido com o objetivo de capacitar jovens estudantes e contribuir positivamente com a comunidade de Amontada através de ações sociais, tecnologia e marketing digital.

---

## 📖 Sobre o projeto

O **Mãos que Ajudam** é uma iniciativa social voltada para o desenvolvimento de jovens, oferecendo oportunidades de aprendizado, capacitação e participação em ações comunitárias.

Além do crescimento profissional dos participantes, o projeto também fortalece vínculos sociais e incentiva a transformação local através da educação e da colaboração.

---

## 🚀 Funcionalidades

* Landing page responsiva
* Sessão sobre o projeto
* Objetivos da iniciativa
* Explicação de funcionamento
* Sessão de impactos sociais
* Galeria interativa com Swiper
* Área de parceiros
* Navegação com scroll suave
* Design moderno e responsivo

## ▶ Como executar o projeto

### 1. Clone o repositório

```bash
git clone maos-que-ajudam
```

---

### 2. Acesse a pasta

```bash
cd maos-que-ajudam
```

---

### 3. Instale as dependências

```bash
npm install
```

---

### 4. Execute o projeto

```bash
npm run dev
```
